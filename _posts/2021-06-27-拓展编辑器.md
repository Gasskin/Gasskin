### 菜单

```csharp
[MenuItem("Assets/MyTools/Tool1",false,2)]
private static void MyTools1()
{
    Debug.Log("right mouse");
}

[MenuItem("Assets/MyTools/Tool2",false,1)]
private static void MyTools2()
{
    Debug.Log("right mouse");
}
```

1. 第一个参数为所在位置，可以添加快捷键，如：

   ```csharp
   _w 单一的快捷键 W 
   #w shift+w 
   %w ctrl+w 
   &w Alt+w
   ```

2. 第二个参数决定了这个方法是否是一个“验证函数”，如果这个方法A是“验证函数”，那么这个方法A会作为另一个**同位置**方法B的先验方法，只有当方法A返回值为true时，才会执行方法B，如下：

   ```csharp
   [MenuItem("GameObject/Extend/Test1",true)]
   private static bool ValidateTest1()
   {
       return Selection.activeObject != null;
   }
   
   [MenuItem("GameObject/Extend/Test1",false)]
   private static void Test1()
   {
       Debug.Log(Selection.gameObjects.Length);
   }
   ```

   你会发现这两个方法的路径名称是一模一样的，而第一个方法的第二个参数为true，说明第一个方法是第二个方法的先验方法，如果是这么写的话，当你没有选中任何物体也就是先验方法返回false时，第二方法会是灰色的，无法点击，只有当你选中某一个物体时，才能够使用第二个方法

3. 第三个参数是排序用的，优先级
