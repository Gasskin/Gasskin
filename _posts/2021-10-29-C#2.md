# 泛型

.NET1中的集合

- **数组** 大小和元素类型在初始化时就必须确定
- **普通集合** 可变长数组，元素均是System.Object类型，如ArrayList，HashTable
- **专用集合** 针对特定类型的集合，如StringCollection，只能保存字符

### 类型形参与类型实参

```c#
public class List<T>{...} // T是类型形参
List<string> list = new List<string> // string是类型实参
```

### 泛型类型和泛型方法

**度** 指泛型声明中类型形参的数量

```c#
public void Mehtod<T1,T2>(){...} // 度为2
```

可以认为非泛型方法的度是0

泛度可以重载泛型方法，但是类型形参不行

```c#
// 编译报错
public void Method<TFirst>(){...}
public void Method<TSecond>(){...}
```

### 类型约束

- **引用约束** where T : class  

  类型实参必须是一个引用类型，包括接口和委托

- **值类型约束** where T : struct

  类型实参必须是非可空值类型，结构体或者枚举

- **构造器约束** where T : new()

  类型实参必须具有公共的无参构造器，保证可以通过new T()创建一个T类型的实例

- **转换约束** where T : SomeType

  类型实参必须继承自指定类型，并且可以转换成指定类型

### default与typeof

default(类型名或类型形参)，返回该类型的默认值，对于引用类型是null；对于非可空值类型，返回该类型的“0值”；对于可空值类型，返回该类型的Null值

typeof不同于GetType()，typeof是一个操作符，针对于某一个类型，而GetType()是一个API，通过具体的实例调用

### 泛型类型初始化与状态

List<int>与List<string>是两个不同的类型，他们之间没有任何关系，不可以进行转换
